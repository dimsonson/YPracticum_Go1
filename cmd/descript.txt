Если в проекте будет несколько бинарных файлов, 
создайте для них поддиректории в cmd. 
Имена поддиректорий должны соответствовать именам исполняемых файлов.
- cmd
    - client
        main.go
    - server
        main.go 
Если в проекте один исполняемый файл, 
исходные файлы можно разместить прямо в директории проекта. 
Старайтесь использовать один-два исходных файла. 
Остальной код нужно вынести в отдельные пакеты.
